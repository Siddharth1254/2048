# Publicity and Sharing

As part of this project, the application was shared within a small and relevant audience to gather initial feedback and test usability.

## How the Project Was Shared

The project was shared with classmates and friends via personal messaging platforms (primarily WhatsApp). Some users were also shown the project directly to guide them through the application and collect immediate reactions.

## Approach

Due to time constraints, the focus was on targeted sharing within our academic and personal network rather than large-scale public outreach. This allowed us to:

- reach users quickly  
- receive meaningful and relevant feedback  
- iterate on improvements in a short time  

## User Interaction

Participants were asked to:

1. Play the game through the live link  
2. Explore the GitHub repository and documentation  
3. Submit feedback through a short Google Form  

## Outcome

This approach resulted in multiple users testing the application and providing useful feedback. Based on this, improvements were made to features such as mobile usability, animations, and overall user experience.

Although the outreach was limited in scale, it was effective in validating the project with real users and identifying areas for improvement.
