# Project 2048

A Flask-based 2048 game with a deployment guide for AWS Elastic Beanstalk.

**Live documentation:** [siddharth1254.github.io/2048](https://siddharth1254.github.io/2048/)

## Repository Structure

```
├── projects/2048-flask-game/   # Flask app source code
├── 2048/                       # Documentation pages (Jekyll)
├── img/                        # Images and assets
├── _config.yml                 # Jekyll configuration
└── index.md                    # Site homepage
```

## Quick Start (Run the Game)

```bash
cd projects/2048-flask-game
pip install -r requirements.txt
python application.py
# Open http://127.0.0.1:5000
```
